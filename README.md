## 🖥️ Using the Workbench

The main window displays your image processing workspace. You can drag images directly into the center area to begin. 

### Processing Images

Use the menu bars at the top to choose your task. 
*   **Face Detection**: Select this option to locate human faces in your pictures.
*   **QR Codes**: Use this to read information from codes.
*   **Image Filters**: Change the look of your files or adjust brightness.

### Applying Changes

After you load an image, select a module from the right side panel. Each module represents a specific visual task. Click the play button to apply the effect. You can see the preview instantly on the screen.

### Saving Results

Select File from the top menu. Click Save As to choose a location for your new file. You can save images in popular formats like JPG or PNG.
